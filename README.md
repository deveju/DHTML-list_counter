# DHTML-list_counter
#### ○ Buttons:
##### Shift + Click increases/decreases the value by 10
##### Ctrl + CLick increases/ decreases the value by 100
##### Click on the + button on bottom right corner to create a new item
### ✦ Every data is saved locally ✦
